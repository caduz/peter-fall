# Peter Fall

A retro pixel-art platformer for casual fun. Jump, climb vines, avoid pits, and collect treasure before time runs out.

![Peter Fall screenshot](screenshot.png)

## Play

👉 [Play in your browser](https://caduz.github.io/peter-fall)

Works on desktop and mobile. On mobile, rotate to landscape for the best experience — on-screen buttons appear automatically.

## Controls

**Desktop**

| Key | Action |
|-----|--------|
| ← → | Move |
| ↑ / Space | Jump |
| ↑ ↓ | Climb vine |
| Enter | Start / Pause |
| M | Mute |

**Mobile**

On-screen ◀ ▶ buttons to move, **B** to jump, and ▶/⏸ to start/pause.

## About

Single HTML file, no dependencies. Old-school arcade vibes.
